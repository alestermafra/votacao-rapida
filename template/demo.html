Votação rápida: Painel de demonstração

Esta é uma votação de demonstração do sistema.
O link Tokens PDF permite baixar os tokens gerados para essa votação.
O apoio controla o andamento da votação que deve ser apresentado pelo painel.
A recepção distribui os QRCodes aos eleitores previamente.

O eleitor recebe um token e um QRCode que dá acesso à interface de votação. Se usar o QRCode, o eleitor entra direto na votação. Se usar o token, ele tem de entrar no link de acesso e digitar o token.

Durante uma votação, é possivel gerar votos aleatórios para fins de demonstração.
Sempre que necessário limpe os dados de votação.
